# ChatGo One-Page Website

This is a static one-page site for ChatGo.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Local Preview

Open `index.html` directly in a browser, or run any static server from this folder.

## Publish Options

### 1) Netlify (quickest)

- Create a new site from drag-and-drop
- Upload the entire `website` folder
- Set custom domain later if needed

### 2) Vercel

- Import this repository
- Set the root/output directory to `website`
- Framework preset: Other

### 3) GitHub Pages

- Push this folder contents to a dedicated branch/repo
- Enable Pages from repository settings

## Before Going Live

- Replace the placeholder Play Store button URL in `index.html` (`id="play-link"`)
- Update metadata/title/description for your final branding
- Optionally add analytics and your privacy policy link
